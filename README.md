## Hosting on GitHub Pages

### Steps to Deploy:

1. **Create a new GitHub repository:**
   - Go to GitHub and create a new repository
   - Name it something like `nabila-congratulations` or `congratulations-nabila`
   - Make it public

2. **Upload your files:**
   ```bash
   cd /data/munna/all/nabila_phd/web
   git init
   git add .
   git commit -m "Initial commit: Congratulations website for Nabila"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

3. **Enable GitHub Pages:**
   - Go to your repository on GitHub
   - Click on "Settings"
   - Scroll down to "Pages" in the left sidebar
   - Under "Source", select "main" branch
   - Click "Save"

4. **Access your website:**
   - Your website will be available at: `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`
   - It may take a few minutes to deploy
